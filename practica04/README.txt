Para compilar el programa debes estar fuera de la carpeta shapesSVG, una vez que estes fuera debes poner el siguiente comando:
 	formasSVG / nombre de la clase que se quiera compilar.java

una vez que haya compilado ahora compilas el Main:
         javac Main.java 

una vez que haya compilado debes correrlo, con el comando java Main 
lo que nos imprimira la información de las figuras en html

Para generar el html debemos de poner el siguiente comando:
          Java Principal> archivo.html

y así es como obtendrás el archivo html con las figuras pedidas.
